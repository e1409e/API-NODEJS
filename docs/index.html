index.js

Configura el servidor Express, establece la conexión con la base de datos PostgreSQL (Neon), configura los middlewares necesarios (CORS, JSON parsing), inyecta la conexión a la base de datos en el objeto de solicitud, y define y monta todas las rutas de la API. También incluye un endpoint de ejemplo para verificar la conexión a la base de datos.
