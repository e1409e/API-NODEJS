Module: routes/estudiantes.routes

Configura los endpoints para realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre los estudiantes. Integra validaciones de datos de entrada utilizando `express-validator` para asegurar la integridad y el formato correcto de la información recibida en las peticiones.
