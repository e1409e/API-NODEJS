Module: routes/carreras.routes

Configura los endpoints para realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre las carreras, enlazando cada ruta con su controlador correspondiente. Las validaciones de los datos de entrada se manejan directamente dentro de los controladores de carrera, utilizando las funciones de validación de `express-validator`.
